# Database Schema (MongoDB)

This document describes the core collections and fields used by HealthFun.

---

## users

| Field | Type | Notes |
|------|------|------|
| _id | ObjectId | primary key |
| username | string | unique |
| passwordHash | string | bcrypt hash |
| name | string | display name |
| avatar | string | avatar URL |
| role | string | Admin / Member / Coach |
| status | string | Active / Suspended |
| createdAt | Date | auto |
| updatedAt | Date | auto |

---

## waterSettings

| Field | Type | Notes |
|------|------|------|
| _id | ObjectId | primary key |
| userId | string | reference to users._id |
| cupSizeCc | number | cup size in cc |
| dailyTargetCc | number | target per day |
| defaultDrinkCc | number | default per entry |
| note | string | optional |
| createdAt | Date | auto |
| updatedAt | Date | auto |

---

## waterEntries

| Field | Type | Notes |
|------|------|------|
| _id | ObjectId | primary key |
| userId | string | reference to users._id |
| date | string | YYYY-MM-DD |
| amountCc | number | water amount |
| time | string | HH:mm |
| note | string | optional |
| createdAt | Date | auto |

---

## tasks

| Field | Type | Notes |
|------|------|------|
| _id | ObjectId | primary key |
| userId | string | reference to users._id |
| date | string | YYYY-MM-DD |
| title | string | task title |
| exp | number | exp points |
| completed | boolean | default false |
| createdAt | Date | auto |
| updatedAt | Date | auto |

---

## logs

| Field | Type | Notes |
|------|------|------|
| _id | ObjectId | primary key |
| userId | string | reference to users._id |
| type | string | food / sport / weight / mood |
| date | string | YYYY-MM-DD |
| time | string | HH:mm |
| data | object | type-specific payload |
| createdAt | Date | auto |
| updatedAt | Date | auto |
